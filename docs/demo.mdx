## Code blocks
### Inline code blocks
You can enclose the inline code in backticks (e.g., `Docuo`) 
```markdown
Add a `inline code block` in a sentence.
```

The rendered output looks like this:
Add a `inline code block` in a sentence.

### Single Code blocks
You can use fenced code blocks by enclosing code in three backticks.
````markdown
```
helloWorld();
```
````
The rendered output looks like this:
```markdown
helloWorld();
```
